Elastos Unity allows third-party
developers to easily create
mobile Elastos Dapps using the
JavaScript language.

Wallet

The Elastos Unity Wallet module adds the SPV wallet native library bindings to your React Native app and therefore allows you to implement all the Elastos wallet functionnalities in your app.

[GitHub]

[Docs]

Carrier

The Elastos Unity Carrier Wallet module adds Carrier library bindings to your React Native app and therefore allows you to implement all the Elastos Carrier p2p networking functionnalities in your app.

[GitHub]

[Docs]

DID (not implemented)

The Elastos Unity DID module will bring all the Elastos DID (Decentralized ID) functionnalities to your React Native app.

HIVE (not implemented)

The Elastos Unity HIVE module will bring all the Elasots HIVE (Distributed storage solution) functionnalities to your React Native app.
